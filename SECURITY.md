# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Ania Creators, please report it responsibly.

**Do not** create a public GitHub issue for security vulnerabilities.

Email: support@aniamodels.shop

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Your contact information

We will respond within 48 hours and work with you to resolve the issue.

## Security Notes

- `.ania` files are server-side validated before playback in Ania Player
- Marketplace uploads go through administrator review before public listing
- Optional password encryption is available for `.ania` exports
- No user telemetry beyond authentication and marketplace transactions
