# Contributing to Ania Creators

Thank you for your interest in Ania Creators!

## What We Accept

- Bug reports
- Feature suggestions
- Documentation improvements
- Translation error reports (194 languages — quality corrections are welcome)

## What We Don't Accept

- Code contributions (source code is proprietary)
- Pull requests modifying core functionality
- Requests to open-source the code

---

## How to Contribute

### Bug Reports

1. Check [existing issues](https://github.com/rob-d3v/ania-creators/issues) to avoid duplicates
2. Open a new issue and include:
   - A clear description of the problem
   - Steps to reproduce
   - Expected vs. actual behavior
   - OS, version, and platform
   - Screenshots or logs if applicable

### Feature Requests

1. Search existing issues first
2. Describe the feature and the use case it solves
3. Explain how it would benefit other users

### Security Issues

**Do not** open public issues for security vulnerabilities.

Email: support@aniamodels.shop — See [SECURITY.md](SECURITY.md) for details.

---

## Other Ways to Contribute

- **Create and publish avatars** on [aniamodels.shop](https://aniamodels.shop) — growing the library benefits the entire community
- **Leave feedback on the marketplace** — ratings and reviews help creators improve
- **Share the project** with people who could use it

---

## Code of Conduct

Be respectful, constructive, and professional in all interactions.
